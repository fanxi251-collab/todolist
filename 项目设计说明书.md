# 项目设计说明书

## 1. 项目概述

Todo List 是一个简洁清爽的日常计划任务管理平台，基于 Python + FastAPI 开发。数据存储在本地 SQLite 数据库，不上传服务端，保护用户隐私。

## 2. 技术栈

### 后端
| 技术 | 版本 | 说明 |
|------|------|------|
| FastAPI | 0.109.0 | Web 框架 |
| Uvicorn | 0.27.0 | ASGI 服务器 |
| SQLAlchemy | 2.0.25 | ORM |
| Pydantic | 2.5.3 | 数据验证 |
| requests | 2.31.0 | HTTP 客户端 |

### 前端
| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构 |
| Tailwind CSS (CDN) | 样式框架 |
| JavaScript (Fetch API) | 交互逻辑 |

### 数据库
- **SQLite**: 轻量级本地文件数据库
- **存储位置**: `todolist.db`

## 3. 项目架构

```
todolist/
├── main.py                 # FastAPI 主入口
├── database.py             # 数据库配置
├── models.py               # SQLAlchemy 模型
├── schemas.py              # Pydantic 数据模型
├── crud.py                 # 数据库操作
├── requirements.txt        # 依赖包
├── routers/                # API 路由
│   ├── notes.py           # 便签 CRUD
│   ├── recycle.py         # 回收站
│   └── weather.py         # 天气查询
├── services/
│   └── weather_service.py # 高德天气 API
├── static/                 # 前端静态文件
│   ├── index.html         # 主页面
│   ├── css/style.css      # 自定义样式
│   └── js/app.js          # 前端逻辑
└── uploads/                # 图片存储目录
```

## 4. 数据模型

### Note (便签)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| content | Text | 便签内容 |
| image_path | String | 图片路径 |
| reminder_date | Date | 提醒日期 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |
| is_deleted | Boolean | 是否删除 |
| deleted_at | DateTime | 删除时间 |

### Tag (标签)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| name | String | 标签名称 |
| color | String | 标签颜色 |

## 5. 接口文档

### 便签管理 `/notes`

| 方法 | 路径 | 说明 | 参数 |
|------|------|------|------|
| GET | /notes | 获取便签列表 | `search`, `tag_id`, `skip`, `limit` |
| GET | /notes/today | 获取今日待办 | - |
| POST | /notes | 创建便签 | `content`, `reminder_date`, `tag_ids`, `image` |
| PUT | /notes/{id} | 更新便签 | `content`, `reminder_date`, `tag_ids`, `image` |
| DELETE | /notes/{id} | 删除便签(移到回收站) | - |
| POST | /notes/export | 导出数据 | `format` (json/csv) |

### 回收站 `/recycle`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /recycle | 获取回收站列表 |
| POST | /recycle/{id}/restore | 恢复便签 |
| DELETE | /recycle/{id} | 彻底删除便签 |
| POST | /recycle/cleanup | 清理过期便签 |

### 天气查询 `/weather`

| 方法 | 路径 | 说明 | 参数 |
|------|------|------|------|
| GET | /weather | 查询天气 | `city`, `days` (默认3天) |

### 标签管理 `/tags`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /tags | 获取标签列表 |
| POST | /tags | 创建标签 |
| DELETE | /tags/{id} | 删除标签 |

## 6. 功能说明

### 天气查询
- 调用高德天气 API
- 支持查询近3天天气预报
- 城市名称支持 (如：北京、上海、广州等)
- 添加重试机制确保稳定性

### 便签记录
- 支持文字和图片
- 设置提醒日期
- 同一日期只能创建一个便签
- 支持标签分类

### 今日待办
- 显示当天日期的便签
- 实时同步便签内容

### 回收站
- 临时存储已删除的便签
- 支持恢复和彻底删除
- 7天自动清理过期便签

### 数据导出
- 支持 JSON 格式导出
- 支持 CSV 格式导出

## 7. 运行方式

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py

# 访问地址
http://127.0.0.1:8000
```
