##### 用Python语言并基于FastAPI框架做一个帮助我记录、管理日常计划任务的界面简洁清爽工具类的todo-list平台。 项目功能专一、操作简单、界面优美，记录只保存在本地，不上传到服务端，不需要担心隐私泄露问题。 前端界面：前端界面有四个小卡片，每个卡片分别对应天气查询，私密便签记录、今日待办功能、回收站功能 【天气查询】 接入天气查询服务，能够知道近三天的天气情况，是否下雨，温度怎么样，这一点可以接入外置服务，请给我推荐 【便签记录】点进去便签记录功能后，可以添加图片和文字到便签中，并且提供日期记录，如果到了那一天，便签所记录的文字会显示在【今日待办】区域，同一天的任务只会存在于一个便签上 【回收站】 可以把不需要的便签放到回收站，并且提供定期删除和主动删除功能，也能够主动撤回，返回到便签中



| **后端框架** | FastAPI                                     | 高性能、自动生成 API 文档，适合快速开发                      |
| ------------ | ------------------------------------------- | ------------------------------------------------------------ |
| **数据库**   | SQLite + SQLAlchemy（ORM）                  | 轻量级本地文件数据库，无需额外安装                           |
| **图片存储** | 本地文件系统（`uploads/` 目录）             | 图片保存到磁盘，数据库只记录路径                             |
| **前端**     | HTML5 + CSS3 + JavaScript（Fetch API）      | 原生三件套，界面清爽，无构建步骤，可直接放在 `static` 目录下 |
| **样式**     | Tailwind CSS 或 手写 CSS                    | 快速实现现代化卡片式 UI，推荐 Tailwind CSS CDN               |
| **天气 API** | 高德天气 API（或 和风天气、OpenWeatherMap） | apikey已配置到环境变量中，名字是：WEATHER_API                |
| **运行环境** | Python 3.8+，在conda环境中运行              | 开发工具：PyCharm                                            |
